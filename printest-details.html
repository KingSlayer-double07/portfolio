PrintEst

A mobile application to help graphic designers calculate total cost of their jobs based on cost of printing, size, medium, etc

React Native TypeScript Expo

Problem

A graphic designer approached me with the challenge of efficiently calculating the total cost of their printing jobs, which involved considering various factors like printing cost, job size, and material type.

Solution

A mobile application designed to streamline the process of calculating the total cost of printing jobs for graphic designers.

Technical Decisions

  • React Native + Expo for cross-platform support and faster iteration
  • TypeScript for type safety and improved development experience

Challenges

One of the main challenges was ensuring the app could accurately calculate costs based on various factors.

Outcome

After lengthy calls with the graphic designer, the app successfully provided an efficient solution for calculating printing costs, saving time and reducing errors.

Future Improvements

  • Add support for many other factors that affect printing costs, such as paper quality, color options, and turnaround time.
  • Add a feature to save and recall previous calculations for quick reference.
  • Include many more print job types that were excluded in the initial version.
  • Implement a user-friendly interface for easier navigation and improved user experience.
